VERTEX RD — Geoportal Catastral 360°

Autor: José Fernando García Pérez
Rol: Senior Geospatial Data Engineer & Analytics Engineer
Ubicación: República Dominicana / Colombia
Fecha: Junio 2026

Resumen ejecutivo: Plataforma web que integra 140,237 predios catastrales del Distrito Nacional con 22 capas GIS, normativa urbanística (densidades y retiros), y tableros analíticos — evolución de un dashboard Power BI y un mapa Mapbox GL JS a un geoportal profesional publicable.

1. Problema de negocio

La planificación urbana del Distrito Nacional requiere consultar simultáneamente: límites catastrales, usos de suelo, restricciones normativas (altura, densidad, retiros), infraestructura vial y espacios públicos. Esta información existía dispersa en shapefiles ArcGIS, tablas Excel y un dashboard Power BI sin acceso web público.

2. Solución implementada

  • Geoportal web con Mapbox GL JS v3.10 (10 tilesets vectoriales)
  • Ficha Predial 360° con búsqueda por código catastro
  • 9 módulos analíticos replicando las pestañas de Power BI
  • Herramientas GIS: mediciones Turf.js, distancias, rutas Directions API
  • Pipeline de datos: Excel → JSON para tablas normativas

Power BI (VERTEX_DN_RD_V3) → KPIs + diseño UI → Dashboard web Chart.js
